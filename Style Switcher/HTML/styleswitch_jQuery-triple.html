3-way jQuery CSS switcher for WCAG/ADA

The click action here cycles through varying degrees of grayscale and contrast. Default theme is normal, WCAG theme is 50% grayscale, and high-contrast theme is 100% grayscale.
There is a toggler floating in the bottom right of the page, a lot of folks do this or up near language selectors.
